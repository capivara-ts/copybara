🐾 Capivara Tech - Copybara Installation

Follow the instructions below to install Copybara on your system:

💻 Linux (Ubuntu / Debian)

Run these commands in a terminal:

curl -fsSL https://capivara-ts.github.io/copybara/pubkey.gpg | gpg --dearmor | sudo tee /usr/share/keyrings/capivara-archive-keyring.gpg >/dev/null
echo "deb [signed-by=/usr/share/keyrings/capivara-archive-keyring.gpg] https://capivara-ts.github.io/copybara/ ./" | sudo tee /etc/apt/sources.list.d/capivara.list
sudo apt update
sudo apt install copybara

🪟 Windows

  • Download the ZIP: Copybara ZIP/Script (rename to copybara.zip if needed)
  • Extract it anywhere, e.g., C:\Users\YourUser\bin\copybara
  • Add that folder to your PATH if needed
  • Open PowerShell or CMD and run: copybara

🍏 macOS

  • Tap the repository and install using Homebrew:
    • brew tap capivara-ts/copybara
brew install copybara
  • Run in Terminal:
    • copybara > text.txt
